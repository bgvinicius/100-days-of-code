Now, using the "clear" property to control the behavior of floats. Check the example below.
Box 1. This box is using a left float. (literally floating..)
Box 2. This element comes after box1, but it isn't floated. We have a small problem here, the box1 is floating above this box... uh-oh.. That happens because the box 1 has a left float. The box2 content is floated around the box1, and the box2 will surround the whole thing.
We can fix this small problem using the "clear" property. Since box1 is floated to the left, we can use clear: left; to clear elements floated to the left, in this case, box1.
Box 1. This box is using a left float.
Box 2. Now, below box1. Using clear: left. We can also use clear: right and clear: both.
When using float, some bad stuff can happen.. Here is an example with an image taller than this element, using a float right. In this example, the image is overflowing outside the div, ouch :(
There is a way to fix this, using the "clearfix" hack. It's kinda ugly, but works with most browsers. Just set this property on the container element: overflow: auto;
Now, there is no overflow, yay! But, kinda ugly :p. Here, we're using overflow: auto; on this div, because it contains the overflowing image. People seems to just create a class called "clearfix" with this property, and then apply it on the necessary elements. More info here.
